Q. 10 - How can we club two or more rows or columns into a single row or column in an HTML table?

  • HTML provides the rowspan and colspan attributes to combine two or more rows or columns into a single row or column in an HTML table.
  • The colspan attribute specifies the number of columns that a cell should span. For example, if you want to merge two cells in the first row into a single cell that spans two columns, you can use the colspan attribute like this:

    • Merged cell Cell 3
    Cell 4 Cell 5 Cell 6

  • In this example, the first cell in the first row spans two columns using the colspan="2" attribute.
  • The rowspan attribute specifies the number of rows that a cell should span. For example, if you want to merge two cells in the first column into a single cell that spans two rows, you can use the rowspan attribute like this:

    • Merged cell Cell 2 Cell 3
    Cell 5 Cell 6

  • In this example, the first cell in the first column spans two rows using the rowspan="2" attribute.
