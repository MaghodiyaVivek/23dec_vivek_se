Q. 14 - What is the use of a span tag? Explain with example?

  • The tag is an HTML tag that is used to group and apply styles to inline elements. It does not have any inherent semantic meaning, but it allows you to apply styles or manipulate a specific part of the text.

    • Here is an example of how to use the tag to apply a different style to a specific part of a sentence:

    This is a < span style="color :red;">red< /span> apple.

    Output : This is a red apple.

  • In this example, the word "red" is wrapped in a tag with a style attribute that sets the color to red. This allows you to apply a different style to that specific word without affecting the rest of the sentence.
