Q. 12 - How to create a Hyperlink in HTML?

  • In HTML, you can create a hyperlink using the < a> (anchor) element. Here's an example of how to create a hyperlink:

    • Click here to visit google.com

    Let's break down this example:

  • The < a> element is used to create the hyperlink.
  • The href attribute specifies the URL that the hyperlink points to. In this example, the URL is "https://www.google.com".
  • The text "Click here to visit google.com" is the anchor text. This is the visible text that the user clicks on to follow the hyperlink.
  • When the user clicks on the hyperlink, they will be taken to the URL specified in the href attribute.
