Q. 13 - What is the use of an iframe tag?

  • The < iframe> tag is an HTML tag used to embed content from another source into a web page. It stands for "inline frame" and allows you to display a separate HTML document within the current web page.
  • The most common use of the < iframe> tag is to embed videos, maps, social media feeds, and other types of content that are hosted on another website. This is useful because it allows you to incorporate dynamic and interactive content into your web page without having to host it yourself.
