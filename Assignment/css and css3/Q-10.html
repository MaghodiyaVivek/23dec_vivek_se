Q. 10 - Why should background and color be used as separate properties?

  • The separation of background and color as separat properties allows for more flexibility and control in web development and design.
  • The background property allows you to set images, patterns, or gradients as the background of an element.
  • The colour property allows you only fill a color in element
  • The background is a Shorthand property, so we can access multiple values in one property
