Q. 3 - What is the difference between CSS2 and CSS3?

  • CSS2 and CSS3 are different versions of the Cascading Style Sheets (CSS) language used for styling web pages.
  • CSS3 introduced some new feachers that offer more flexibility on a web page.
  • CSS3 introduced new properties and values for styling backgrounds and borders.
  • CSS3 introduced new properties for text styling, such as text-shadow, and text-justify.
