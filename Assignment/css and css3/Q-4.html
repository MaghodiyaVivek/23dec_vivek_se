Q. 4 - Name a few CSS style components

  • Here are a few CSS style components commonly used in web development:
  • Color: CSS allows you to specify colors for text, backgrounds, borders, and other elements using color names, hexadecimal values, or RGB values
  • Backgrounds: CSS allows you to set background colors, images, gradients, and positions for elements. Properties like background-color, background-image, background-repeat, and background-position are used to control backgrounds.
  • Borders: CSS provides properties to control the appearance of borders around elements. You can specify border width, style (solid, dashed, etc.), and color using properties like border-width, border-style, and border-color.
