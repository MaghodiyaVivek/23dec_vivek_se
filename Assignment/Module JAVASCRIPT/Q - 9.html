Q.9 How can the style/class of an element be changed?

  • In JavaScript, you can change the style or class of an HTML element using the style property and the classList property.
  • The style property allows you to manipulate individual CSS properties of an element directly.
  • You can set or modify specific style attributes like color, background, font-size, etc.
  • The classList property is used to manipulate the classes applied to an element. You can add, remove, or toggle classes using this property.
