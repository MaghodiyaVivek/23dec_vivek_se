Q.8 What is === operator?

  • === is a strict equality comparison operator in javascript Which returen false for the value which are not of a similar type.
  • strict equality === checks that two value are the same or not.
  • Value are not implicity converted to some other value before comparison.
  • This operator performs type casting for equality.if we compare 2 with 2 using ===, then it will return a false value.
