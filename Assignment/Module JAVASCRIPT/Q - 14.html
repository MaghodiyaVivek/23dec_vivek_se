Q.14 What are all the types of Pop up boxes available in JavaScript?

  • In JavaScript, there are three main types of popup boxes that you can use to interact with users: alert, prompt, and confirm.
  • These are built-in functions provided by the browser's JavaScript runtime for displaying simple dialog boxes to the user.

  • Alert Box:
  • The alert() function displays a simple message box to the user with an "OK" button.
  • It is often used to provide information or to give a warning to the user. The user can only dismiss the alert by clicking the "OK" button.

  • Prompt Box:
  • The prompt() function displays a message box to the user with an input field where they can enter text.
  • It's commonly used to get user input.

  • Confirm Box:
  • The confirm() function displays a message box to the user with two buttons: "OK" and "Cancel".
  • It's used to get a confirmation from the user.
  • The function returns true if the user clicks "OK" and false if the user clicks "Cancel".
